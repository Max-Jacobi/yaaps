# Yet Another Athena Plot Script (YAAPS)

Postprocessing scripts for (GR)athena++

## Documentation

| Module | Description |
|--------|-------------|
| [Simulation](docs/simulation.md) | Main entry point — load data, create plots, access history/waveform/tracer/horizon files |
| [Data Types](docs/datatypes.md) | `Native`, `Derived`, `Vector` mesh-data loaders and interpolation |
| [2D Plotting](docs/plot2D.md) | Colour plots, quiver, streamline, contour, tracer scatter, animations |
| [Plot Formatter](docs/plot_formatter.md) | Raw vs. paper mode — labels and unit conversions |
| [Units](docs/units.md) | `UnitConverter` and `FieldLabels` for physical units and LaTeX labels |
| [Decorations](docs/decorations.md) | Default colormap/normalization settings and variable aliases |
| [Recipes 2D](docs/recipes2D.md) | Metric tensor utilities (det, inverse, raise/lower, radial projection) |
| [Input](docs/input.md) | Athena++ parameter file parser with diff support |
| [Parallel Utilities](docs/parallel_utils.md) | Parallel task execution with progress bars |
| [Examples](docs/examples.md) | CLI example scripts and programmatic usage snippets |
